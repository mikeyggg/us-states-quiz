US States Quiz 🗺️
Interactive game built with Python and Turtle where you guess all 50 U.S. states.
How to play
Type the name of a U.S. state in the input box. If correct, it will appear on the map. Try to guess all 50!
Requirements

Python 3
pandas

How to run
pip install pandas
python main.py
Files

main.py — game logic
move_text.py — turtle class
get_cords.py — loads state coordinates from CSV
